# Introduction

[Planit](https://planit.simplyblackmedia.com) is a media and advertisement platform that I led through a comprehensive legacy platform rewrite. As tech lead for a team of 5 developers, I designed the application architecture, enforced peer code reviews, and integrated AWS QuickSight dashboards for real-time business intelligence.

## Key Capabilities

- **Legacy Modernization**: Led a complete rewrite of a legacy media platform, modernizing the tech stack and architecture while maintaining business continuity.
- **Business Intelligence**: Integrated AWS QuickSight dashboards providing real-time advertising performance metrics, campaign analytics, and revenue reporting.
- **Code Quality**: Established mandatory peer code reviews, automated linting, and testing standards that raised engineering quality across the team.

## Technical Insights

- **Frontend**: React for the new application interface, replacing legacy Angular components incrementally.
- **Backend**: Node.js API layer with Python services for data processing and analytics pipelines.
- **BI & Analytics**: AWS QuickSight for interactive dashboards with embedded analytics directly within the application UI.
- **Infrastructure**: AWS-hosted with CI/CD pipelines for automated deployment.

## Challenges and Solutions

- **Legacy Migration Risk**: Rewriting a live platform without downtime required careful phasing. Adopted a strangler pattern — gradually routing traffic from legacy to new services while maintaining full backward compatibility.
- **Team Ramp-Up**: With 5 developers, ensured consistent code quality through structured peer reviews, shared architecture documentation, and pair programming sessions during the initial phase.

## Impact

Successfully migrated a legacy media platform to a modern, scalable architecture. The embedded QuickSight dashboards gave stakeholders real-time visibility into advertising performance, replacing manual reporting processes that previously took days to compile.
