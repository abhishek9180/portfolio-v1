# Introduction

theCompass is a cloud-native values assessment application I built from inception alongside one developer. Starting from ambiguous client requirements, I formulated the complete systems design, database schemas, and end-to-end tech stack.

## Key Capabilities

- **Values Assessment & Discovery Engine**: Developed a highly responsive, interactive assessment flow leveraging React and Framer Motion, enabling complex multi-step user inputs to map and compute personal value profiles dynamically.
- **Full Ownership**: From ambiguous client requirements, designed the complete system architecture, data models, and infrastructure blueprint.
- **Cloud-Native Infrastructure**: Created the entire cloud infrastructure using AWS Infrastructure as Code (IaC), eliminating manual environment configurations and preventing configuration drift.
- **Small Team**: Led end-to-end development alongside one developer, establishing engineering practices and code quality standards.

## Technical Insights

- **Backend**: NestJS for a modular, testable API layer with TypeScript throughout.
- **Frontend**: React with Tailwind CSS for rapid, consistent UI development.
- **Infrastructure**: AWS (IaC) for compute, storage, and networking — fully codified and version-controlled.
- **Database**: Custom schema design optimized for the assessment domain's query patterns.

## Challenges and Solutions

- **Ambiguous Requirements**: The client had a high-level vision but no detailed specifications. Conducted iterative discovery sessions and translated findings into concrete system designs and prototypes.
- **Single-Developer Risk**: With only two developers, established clear ownership boundaries, comprehensive documentation, and automated testing to maintain velocity without sacrificing quality.

## Impact

Delivered a fully functional cloud-native application, with all infrastructure codified as reusable IaC templates. The project demonstrated the ability to take ambiguous concepts and translate them into production-ready systems.
